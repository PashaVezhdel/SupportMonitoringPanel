# 🖥️ Support Monitoring Panel
Система моніторингу та аналітики заявок служби технічної підтримки в реальному часі. Проєкт розроблений як частина дипломної роботи для автоматизації контролю роботи IT-сервісу та візуалізації ефективності обробки інцидентів.

## 🚀 Основні можливості
* Dashboard 2x2: Візуалізація ключових метрик через динамічні графіки Chart.js.
* Real-time оновлення: Отримання свіжих даних з бази без перезавантаження сторінки через API.
* Фільтрація: Гнучкий вибір періоду дати Від та До для аналізу заявок.
* Живий пошук: Пошук у списку заявок за ключовими словами ID, ім'я, статус, опис.
* Мобільна адаптивність: Коректне відображення на різних пристроях завдяки Bootstrap 5.

## 🛠 Стек технологій
* Backend: Python 3.x, Django 5.x
* Database: MongoDB NoSQL — збереження заявок
* Frontend: HTML5, CSS3, JavaScript ES6+, Bootstrap 5
* Charts: Chart.js — побудова аналітики
* Infrastructure: Docker, Docker Compose контейнеризація 🐳

## 📂 Структура проєкту
* config/ — основні налаштування Django-проєкту.
* dashboard/ — логіка застосунку views, templates.
* static/ — розділені файли стилів CSS та скриптів JS.
* requirements.txt — список усіх залежностей.
* Dockerfile — конфігурація збірки контейнера.
* docker-compose.yml — оркестрація веб-сервісу та бази даних.
